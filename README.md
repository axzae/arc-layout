This is an **Kotlin implementation** of [florent37's arclayout](https://github.com/florent37/arclayout).

By using this version, you can:
- use MavenCentral

## Usage

#### Gradle

```kotlin
// build.gradle.kts (app module)

dependencies {
    implementation("com.axzae:arc-layout:1.1.0")
}
```
